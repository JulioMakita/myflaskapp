Solve Python Pip error

If you get the following error when try to use a PIP code:

Could not install packages due to an EnvironmentError: [Errno 13] Permission denied: '/usr/local/lib/python2.7/dist-packages' Consider using the `--user` option or check the permissions.
You are using pip version 10.0.1, however version 18.1 is available. You should consider upgrading via the 'pip install --upgrade pip' command.

Use this command: pip install --user tensorflow
AND sudo chmod a+w /usr/local/lib/python2.7/dist-packages to give a permission

Remove PIP for python2 = sudo apt-get remove python-pip
